Sound Effects for Quiz Application

To enable sound effects in the app, add the following files to this directory:

1. correct.mp3 - Sound to play when a user selects the correct answer
2. incorrect.mp3 - Sound to play when a user selects an incorrect answer

You can find free sound effects from sources like:
- freesound.org
- pixabay.com/sound-effects/
- mixkit.co/free-sound-effects/

Simply download the files, rename them to correct.mp3 and incorrect.mp3, and place them in this directory.
